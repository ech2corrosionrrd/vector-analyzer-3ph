# CHANGELOG - VectorAnalyzer 3Ph

Всі зміни та покращення внесені під час сесії оновлення проєкту.

## [2026-03-20] - Оновлення та Оптимізація

### Додано
- **Конфігурація PWA**: Додано `vite-plugin-pwa` у `vite.config.js` для підтримки офлайн-режиму та інсталяції на смартфони.
- **Діагностика**: Покращено логіку визначення чергування фаз та перевернутої полярності.
- **Автоматичне розгортання**: Створено GitHub Actions workflow (`.github/workflows/deploy.yml`) для автоматичної публікації на GitHub Pages.
- **Інструкції**: Додано файли `installation_guide.md` та `README.md` з актуальними кроками по встановленню.
- **Git**: Ініціалізовано локальний репозиторій та зроблено перший коміт.

### Змінено
- **Метадані**: Placeholder-назви "-----" замінено на професійну назву `VectorAnalyzer 3Ph`.
- **Експорт PDF**: Оновлено функцію експорту — кнопки керування тепер автоматично ховаються у згенерованому звіті.
- **Стилізація**: Покращено візуальну ієрархію та акценти на векторній діаграмі.

### Виправлено
- Виправлено структуру папок для коректної роботи GitHub Pages (вимоги до розміщення `.github`).
- Усунено проблеми з відображенням SVG-маркерів при експорті (фільтр у `toPng`).
