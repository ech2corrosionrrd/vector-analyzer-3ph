# VectorAnalyzer 3Ph - Інструкція зі встановлення

Професійний вебдодаток для візуалізації та аналізу векторних діаграм трифазних мереж.

## 1. Початкові вимоги
Для запуску додатка на комп'ютері повинен бути встановлений **Node.js** (версія 18 або новіша).
Завантажити можна тут: [nodejs.org](https://nodejs.org/)

## 2. Встановлення та запуск на комп'ютері

1. **Завантажте проєкт** (якщо ви отримали архів, розархівуйте його в окрему папку).
2. **Відкрийте термінал** (PowerShell або CMD) у папці з проєктом.
3. **Встановіть залежності**:
   ```bash
   npm install --legacy-peer-deps
   ```
4. **Запустіть додаток у режимі розробки**:
   ```bash
   npm run dev
   ```
5. Після запуску термінал покаже посилання (зазвичай `http://localhost:5173`). Відкрийте його у браузері.

## 3. Встановлення як додатка (PWA)

Додаток підтримує технологію **PWA**, що дозволяє встановити його на смартфон або комп'ютер як окрему програму, яка працюватиме навіть без інтернету.

### На Android (Chrome):
1. Відкрийте посилання на додаток.
2. Натисніть на три крапки (меню) у верхньому правому куті.
3. Виберіть **"Додати на головний екран"** або **"Встановити додаток"**.

### На iPhone/iPad (Safari):
1. Відкрийте посилання у Safari.
2. Натисніть кнопку **"Поділитися"** (квадрат зі стрілкою вгору).
3. Прокрутіть вниз і виберіть **"Додати до початкового екрана"**.

### На комп'ютері (Chrome/Edge):
1. В адресному рядку браузера з'явиться іконка (монітор зі стрілкою).
2. Натисніть на неї та виберіть **"Встановити"**.

## 4. Створення звіту у PDF
У верній частині екрана є кнопка **"Експорт PDF"**. Вона автоматично генерує звіт з діаграмою та таблицями результатів. Звіт оптимізовано для друку на аркуші A4.

## 5. Основні гарячі клавіші та поради
- **Зміна кутів**: Ви можете перемикатися між режимом "Відносно UA" (класична полярна діаграма) та "Кут φ" (зручно для перевірки актів).
- **Кольори**: Жовтий - Фаза А, Зелений - Фаза B, Червоний - Фаза C.
- **Діагностика**: Якщо система бачить червону рамку в результатах — це критична помилка (наприклад, перевернута фаза).
